9.What are CSS combinators 

    There are four types of combinators we use in CSS: 

    1. space ( ): to select the descedents
    2. greater than symbol (>): to select the children only
    3. plus sign (+): to select the imidiate sibling
    4. tilda (~): to select all the next siblings under the same parent


10.What are pseudo selectors in CSS

    pseudo-class selectors are such as:

    :hover --> gets activated when cursor is hovered about the selected element or class
    :visited --> it selects the link which have already been visited by the user on the same page
    :active --> gets activated when we click or select that element or class
    :focus --> focus on the currently select input or any element
    :enabled --> to enable the selected 
    :Disable --> to disable the selected
